Discord Settings

Erstelle eine Discord-Anwendung für den Bot. Rufe das Discord Developer Portal auf und melde dich an. Erstelle eine neue Anwendung und:

  1. Go to "Bot" in the sidebar and click "Add Bot"
  2. In the "Bot" section, scroll down to "Privileged Gateway Intents" and enable SERVER MEMBERS INTENT (required for user mappings)
  3. Go to "OAuth2" → "URL Generator"
  4. Select scopes: bot and applications.commands
  5. Select permissions: Send Messages and Embed Links
  6. Copy the generated URL, open it in your browser, and authorize the bot on your Discord server
  7. Come back and enter the Discord Bot Token and Bot Client ID below, save your settings, start the bot, then select your server and channel from the dropdowns

About Questorr

Questorr is a self-hosted bridge between your Jellyfin media server and Discord. It connects Jellyfin and Seerr so your community can search, request and track media — all without leaving Discord.

Features

  • /search <title> Search for movies and TV shows with auto-suggestions. A rich embed with poster, description, rating and genre is shown — request directly from the embed. For TV shows, you can select individual seasons.
  • /request <title> Instantly send a media request to Seerr. For TV shows, all seasons are requested automatically. Optionally configure tag, server and quality profile selection.
  • /trending Browse the week's most popular movies and TV shows from TMDB — discover what's hot right now and request directly.
  • /status <title> Check the current Seerr request status of any title in Discord — with poster, summary, genre, runtime, rating and age rating. Shows a Request button if not yet requested.
  • /random movie or /random series — get a random suggestion from your Jellyfin library with poster, summary, genre, runtime, rating and Watch Now button.
  • Receive rich Discord notifications for every Seerr event: pending approval, approved, available, declined, failed and issues — each routed to the right channel automatically.
  • Automatically route MEDIA_AVAILABLE notifications to the correct Discord channel based on the Radarr/Sonarr root folder — one channel per library category.
  • Embeds are enriched with data from TMDB and OMDb: posters, backdrops, IMDb ratings, runtime, genre and a direct Watch Now link.
  • Control who can use bot commands via Discord role allowlist and blocklist. Link Discord accounts to Seerr accounts so every request is tracked to the correct user.
  • Users receive a private DM when their requested content becomes available on Jellyfin — so nobody misses the moment their request is ready.
  • Post a Daily Random Pick from TMDB or a Daily Recommendation from your existing Jellyfin library into a Discord channel — great for community engagement.
  • Configure everything through the web dashboard at port 8282. Supports English and German, is fully mobile-responsive and works on smartphones and tablets.
